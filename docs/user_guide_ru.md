## 1. Начало работы

### 1.1 Регистрация

1. Откройте приложение и перейдите на страницу входа
2. Нажмите **«Регистрация»**
3. Заполните поля:
   - **Email** — ваш рабочий email
   - **Пароль** — минимум 8 символов, включая цифры и спецсимволы
   - **Полное имя** — ваше имя для отображения
4. Нажмите **«Создать аккаунт»**
5. Вы будете автоматически перенаправлены на дашборд

### 1.2 Вход в систему

1. Введите email и пароль
2. Нажмите **«Войти»**
3. Если включена двухфакторная аутентификация (MFA), введите 6-значный код из приложения-аутентификатора

### 1.3 Интерфейс — обзор навигации

После входа вы увидите основной дашборд с боковым меню:

- **Дашборд** — KPI-показатели, графики, курсы валют, последние операции
- **Портфели** — управление инвестиционными портфелями
- **Решения** — инвестиционные решения (создание, анализ, версионирование)
- **AI-аналитика** — AI-powered анализ и прогнозирование
- **Due Diligence** — скоринг и проверка компаний
- **Калькулятор** — финансовые расчёты (DCF, NPV, IRR, WACC)
- **Исламские финансы** — шариатский скрининг и закят
- **Бизнес-кейсы** — 50+ бизнес-кейсов с валидацией
- **Настройки** — профиль, тема, язык, уведомления

### 1.4 Настройка профиля

1. Перейдите в **Настройки → Профиль**
2. Обновите имя при необходимости
3. Для смены пароля введите текущий пароль и новый дважды
4. Нажмите **«Сохранить»**

## 2. Портфели

### 2.1 Создание портфеля

1. Перейдите в раздел **«Портфели»**
2. Нажмите **«Создать портфель»**
3. Заполните поля:
   - **Название** — например, «Агротех Узбекистан»
   - **Описание** — краткое описание стратегии
   - **Начальная стоимость** — в UZS
4. Нажмите **«Создать»**

### 2.2 Добавление активов

1. Откройте портфель, нажав на его карточку
2. Нажмите **«Добавить решение»**
3. Заполните параметры инвестиционного решения:
   - **Актив** — название компании/актива
   - **Тикер** — символ актива
   - **Тип** — Покупка (BUY), Продажа (SELL), Удержание (HOLD)
   - **Объём** — количество единиц
   - **Цена** — цена за единицу в UZS
   - **Приоритет** — низкий, средний, высокий, критический
   - **Категория** — акции, долг, недвижимость, венчур
   - **Обоснование** — детальное обоснование решения
4. Нажмите **«Сохранить»**

### 2.3 Анализ портфеля

На странице портфеля доступны:
- **Общая стоимость** — сумма всех активов
- **Распределение по секторам** — круговая диаграмма
- **История изменений** — временная линия
- **AI-рекомендации** — автоматический анализ от AI

Для запуска Monte Carlo симуляции:
1. Нажмите **«Monte Carlo анализ»**
2. Выберите сектор и параметры
3. Получите распределение вероятностей доходности

## 3. AI-аналитика

### 3.1 Запрос AI-анализа

1. Перейдите в **«AI-аналитика»**
2. Введите запрос на естественном языке, например:
   - «Проанализируй перспективы IT-сектора Узбекистана»
   - «Оцени риски инвестиций в агротехнологии»
   - «Сравни ROI строительного и торгового секторов»
3. Нажмите **«Анализировать»**
4. AI обработает запрос через несколько провайдеров и вернёт обобщённый результат

### 3.2 Понимание результатов (XAI)

Система XAI (Explainable AI) объясняет каждое AI-решение:

- **Факторы влияния** — какие факторы повлияли на рекомендацию
- **Важность факторов** — вес каждого фактора (визуализация)
- **Доверительный интервал** — уровень уверенности AI
- **Секторальные корректировки** — учёт специфики сектора

Для запуска XAI-анализа:
1. Перейдите в **«XAI Анализ»**
2. Укажите сектор, сумму инвестиций и горизонт
3. Получите детальную разбивку по факторам

### 3.3 Сценарный анализ

Monte Carlo v2 калиброван под экономику Узбекистана:

- **Сельское хозяйство** — средняя доходность 18%, волатильность 35%
- **Пищевая промышленность** — доходность 20%, волатильность 28%
- **Торговля** — доходность 22%, волатильность 22%
- **Строительство** — доходность 25%, волатильность 30%
- **Промышленность** — доходность 20%, волатильность 28%
- **IT и услуги** — доходность 30%, волатильность 40%

Результаты включают: перцентили (P10, P25, P50, P75, P90), VaR, Expected Shortfall.

## 4. Due Diligence

### 4.1 Поиск компании

1. Перейдите в **«Due Diligence»**
2. Введите название компании или ИНН в строку поиска
3. Система найдёт компанию в базе данных

### 4.2 DD-скоринг

1. Нажмите **«Запустить DD-скоринг»**
2. Укажите параметры:
   - **Отрасль** — для подбора бенчмарков
   - **География** — страна (UZ, KZ и др.)
   - **Финансовые показатели** — выручка, активы
3. Система проведёт скоринг по критериям:
   - Финансовая устойчивость
   - Юридическая чистота
   - Рыночная позиция
   - Управленческая команда
   - Комплаенс
4. Результат — общий балл (0-100) и детализация по категориям

### 4.3 Загрузка документов

1. На странице DD нажмите **«Загрузить документ»**
2. Поддерживаемые форматы: PDF, DOCX, XLSX
3. AI автоматически проанализирует документ и добавит информацию к DD-отчёту
4. Красные флаги будут выделены автоматически

## 5. Калькулятор

### 5.1 DCF модель

DCF (Discounted Cash Flow) — оценка стоимости компании:

1. Перейдите в **«Калькулятор» → DCF**
2. Введите:
   - **Денежные потоки** — прогноз по годам (CF0 — инвестиция, отрицательная)
   - **Ставка дисконтирования** — например, 12%
   - **Терминальный рост** — долгосрочный рост (2-3%)
3. Нажмите **«Рассчитать»**
4. Получите DCF-оценку стоимости

### 5.2 NPV/IRR расчёты

**NPV (Net Present Value):**
- Введите денежные потоки и ставку дисконтирования
- NPV > 0 — инвестиция выгодна

**IRR (Internal Rate of Return):**
- Введите денежные потоки (первый — отрицательный)
- IRR > требуемой доходности — инвестиция приемлема

### 5.3 WACC

WACC (средневзвешенная стоимость капитала):

1. Введите:
   - **Собственный капитал** — объём в UZS
   - **Заёмный капитал** — объём в UZS
   - **Стоимость собственного капитала** — %
   - **Стоимость заёмного капитала** — %
   - **Ставка налога** — 15% по умолчанию для УЗ
2. Получите WACC — используйте как ставку дисконтирования в DCF

## 6. Исламские финансы

### 6.1 Шариатский скрининг

Проверка компании на соответствие нормам шариата:

1. Перейдите в **«Исламские финансы» → Скрининг**
2. Введите:
   - Название компании
   - Выручка
   - Общий долг
   - Общие активы
   - Доход от не-халяльной деятельности
3. Система проверит по критериям AAOIFI:
   - Соотношение долга к активам (< 33%)
   - Доля не-халяльного дохода (< 5%)
   - Процентные доходы (< 5%)
4. Результат: **Халяль** или **Не-халяль** с пояснениями

### 6.2 Закят-калькулятор

Расчёт обязательного закята:

1. Перейдите в **«Исламские финансы» → Закят»**
2. Введите:
   - **Общие активы** — накопления, инвестиции, золото
   - **Обязательства** — долги, кредиты
3. Система рассчитает:
   - Закят-базу (активы - обязательства)
   - Сумму закята (2.5% от закят-базы)
   - Проверку нисаба (минимальный порог)

## 7. Экспорт отчётов

### 7.1 PDF с брендированием

Генерация профессиональных PDF-отчётов:

1. На странице портфеля нажмите **«Экспорт PDF»**
2. Выберите тип отчёта:
   - **Сводка по портфелю** — общий обзор, KPI, графики
   - **DD-отчёт** — результаты Due Diligence
   - **Меморандум решения** — детали инвестиционного решения
3. PDF включает:
   - Логотип AI Capital Management
   - Дату генерации
   - Все ключевые метрики
   - AI-рекомендации

### 7.2 Excel с формулами

Экспорт данных в формате Excel:

1. Нажмите **«Экспорт Excel»**
2. Выберите тип:
   - **Портфель** — список активов, аналитика, формулы
   - **DD-отчёт** — скоринг, чеклисты, комментарии
   - **Сравнение компаний** — табличное сравнение
3. Excel файл содержит:
   - Активные формулы для пересчёта
   - Условное форматирование
   - Графики

## 8. Настройки

### 8.1 Тёмная тема

1. Перейдите в **Настройки → Оформление**
2. В разделе **«Тема»** выберите:
   - **Светлая** — стандартная тема
   - **Тёмная** — для комфортной работы в темноте
   - **Системная** — следует настройкам ОС

### 8.2 Язык интерфейса

1. В разделе **«Оформление»** выберите язык:
   - **Русский** — основной язык
   - **O'zbek** — узбекский
   - **English** — английский
2. Интерфейс переключится мгновенно

Дополнительные настройки оформления:
- **Валюта** — UZS, USD, EUR, RUB
- **Часовой пояс** — UTC+5 Ташкент по умолчанию
- **Формат даты** — ДД.ММ.ГГГГ, ММ/ДД/ГГГГ, ГГГГ-ММ-ДД
- **Размер шрифта** — мелкий, обычный, крупный

### 8.3 MFA (двухфакторная аутентификация)

1. Перейдите в **Настройки → Безопасность**
2. Нажмите **«Настроить MFA»**
3. Отсканируйте QR-код приложением-аутентификатором (Google Authenticator, Authy)
4. Введите 6-значный код для подтверждения
5. MFA будет запрашиваться при каждом входе

## Часто задаваемые вопросы

**В: Как сбросить пароль?**
О: На странице входа нажмите «Забыли пароль?» и следуйте инструкциям.

**В: Как удалить портфель?**
О: Откройте портфель → кнопка «Удалить» → подтвердите удаление.

О: Система поддерживает Groq, Gemini, OpenAI, Ollama, HuggingFace. Оркестратор автоматически выбирает оптимальный провайдер.

О: На любой странице с данными нажмите кнопку «Экспорт» и выберите формат (PDF или Excel).

**В: Безопасны ли мои данные?**
О: Все данные зашифрованы, доступ по JWT-токенам, поддержка MFA, аудит действий.

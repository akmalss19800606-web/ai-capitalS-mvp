## 3. Портфельный анализ (3 минуты)

### Действия
1. **Открыть портфели**
   - Показать 5 портфелей: «Агротех Узбекистан», «IT Park Стартапы», «Торговый Дом», «Исламские Инвестиции», «Диверсифицированный»

2. **Открыть «Агротех Узбекистан»**
   - Показать инвестиционные решения: Cotton Processing, Agro Fruit Export
   - Обратить внимание на суммы (в UZS), статусы (approved, review, draft)
   - Показать обоснование каждого решения

3. **Monte Carlo симуляция**
   - Перейти в AI-аналитику или Monte Carlo
   - Запустить симуляцию для сектора «Сельское хозяйство»
   - Параметры: 1 000 000 UZS, 5 лет, 1000 симуляций
   - Показать результаты: распределение доходности, перцентили (P10, P50, P90), VaR

4. **XAI — Объяснимость**
   - Перейти в XAI анализ
   - Запустить анализ для «agriculture» сектора
   - Показать факторы влияния и их веса
   - Подчеркнуть: «AI объясняет каждое решение — это не чёрный ящик»

### Тезис для инвестора
> «Наша система не просто даёт рекомендации — она объясняет ПОЧЕМУ. Каждый фактор визуализирован и взвешен. Инвестор контролирует процесс.»

## 4. Due Diligence (2 минуты)

### Действия
1. **Поиск компании**
   - Ввести: «UzAuto Motors» или ИНН
   - Показать результаты поиска

2. **DD-скоринг**
   - Запустить скоринг: отрасль «automotive», география «UZ»
   - Показать результаты:
     - Общий балл (0-100)
     - Детализация по категориям: финансы, юридика, рынок, управление
     - Бенчмарки отрасли

3. **Загрузка документа** (опционально)
   - Показать интерфейс загрузки (PDF, DOCX)
   - Объяснить: «AI автоматически анализирует документ и выявляет красные флаги»

### Тезис для инвестора
> «Due Diligence за минуты вместо недель. AI-скоринг с отраслевыми бенчмарками для Узбекистана.»

---

## 5. Калькулятор (1 минута)

### Действия
1. **DCF расчёт**
   - Ввести реальный кейс:
     - Инвестиция: -2 000 000 000 UZS
     - Потоки: 600M, 700M, 800M, 900M, 1 000M
     - Ставка: 12%
   - Показать результат DCF-оценки

2. **IRR/NPV**
   - Показать: NPV положительный → инвестиция выгодна
   - IRR выше WACC → доходность приемлема

### Тезис для инвестора
> «Профессиональный инвестиционный калькулятор с DCF, NPV, IRR, WACC — всё в одном месте.»

---

## 6. Исламские финансы (1 минута)

### Действия
1. **Шариатский скрининг**
   - Выбрать компанию для проверки
   - Показать результат: Халяль / Не-халяль с пояснениями
   - Критерии AAOIFI: долг < 33%, не-халяль доход < 5%

2. **Закят-калькулятор**
   - Ввести: активы 500M UZS, обязательства 100M UZS
   - Показать расчёт закята (2.5% от закят-базы)

### Тезис для инвестора
> «Единственная инвестиционная AI-платформа в Центральной Азии с поддержкой исламских финансов. Рынок халяль-инвестиций растёт на 25% в год.»

## Ответы на возможные вопросы

**В: Какой стек технологий?**
О: FastAPI + PostgreSQL + Next.js 16 + Redis. Docker-контейнеры для деплоя.

**В: Как обеспечивается безопасность?**
О: JWT-аутентификация, MFA (TOTP), rate limiting, аудит логи, шифрование.

**В: Какие AI-модели используются?**
О: Мульти-провайдерная архитектура: Groq (LLaMA), Google Gemini, OpenAI GPT, Ollama (локальный). Circuit breaker для отказоустойчивости.

**В: Как насчёт масштабирования?**
О: Архитектура поддерживает горизонтальное масштабирование. Redis для кэширования. Асинхронные AI-запросы.

**В: Есть ли мобильная версия?**
О: Адаптивный дизайн — работает на смартфонах и планшетах. Нативное приложение в роадмапе.

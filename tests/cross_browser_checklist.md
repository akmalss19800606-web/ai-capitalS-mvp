## Известные CSS-особенности и фиксы

### Safari
- `backdrop-filter` требует `-webkit-backdrop-filter` (уже поддержан Tailwind v4)
- `position: sticky` внутри `overflow: auto` контейнеров может не работать
- `gap` в flexbox полностью поддерживается с Safari 14.1+

### Firefox
- `scrollbar-width: thin` вместо `::-webkit-scrollbar`
- `appearance: none` для кастомных select-элементов
- `backdrop-filter` поддерживается с Firefox 103+

### Edge
- Идентичен Chrome (Chromium-based), минимальные отличия
- Старый Edge (EdgeHTML) не поддерживается

## Применённые CSS-фиксы

1. **Кроссбраузерные скроллбары**: используется стандартный `scrollbar-width` + `-webkit-scrollbar` fallback
2. **Backdrop filter**: `-webkit-backdrop-filter` добавлен через Tailwind v4 утилиты
3. **Font rendering**: `-webkit-font-smoothing: antialiased` + `-moz-osx-font-smoothing: grayscale`
4. **Input styling**: `appearance: none` для унификации стилей форм

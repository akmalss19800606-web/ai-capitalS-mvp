## 8. Применённые CSS-фиксы для мобильных

### Рекомендации по responsive-стилям

1. **Таблицы**: `overflow-x: auto` + `min-width` на таблице для горизонтальной прокрутки
2. **Grid-layout**: `grid-template-columns: 1fr` на мобильных вместо multi-column
3. **Графики Recharts**: использовать `ResponsiveContainer` с `width="100%"` и `height={250}`
4. **Кнопки**: `min-height: 44px` + `padding: 12px 16px` для touch-friendly
5. **Модалки**: `max-height: 90vh` + `overflow-y: auto` для мобильных
6. **Навигация**: sticky hamburger с `position: fixed; z-index: 50`

### CSS для адаптивных таблиц

```css
/* Обёртка таблицы с горизонтальной прокруткой */
.table-responsive {
  overflow-x: auto;
  -webkit-overflow-scrolling: touch;
}

/* На мобильных — уменьшенный padding */
@media (max-width: 640px) {
  .table-responsive td,
  .table-responsive th {
    padding: 8px 12px;
    font-size: 13px;
    white-space: nowrap;
  }
}
```

### CSS для touch-friendly элементов

```css
/* Минимальный размер touch-target */
@media (max-width: 768px) {
  button, a, [role="button"] {
    min-height: 44px;
    min-width: 44px;
  }
}
```
